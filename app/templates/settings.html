    Keyboard bindings

    Search files alt P
    Close tab alt W
    Surround selection in LaTeX command alt C
    Jump to file under cursor alt G
    Set current tab as main tab alt M
    Set current tab as a favorite tab alt D

    Bugs, questions & feature requests

    If you encounter any bugs please do report them on github at the SLext issue tracker. For support and questions you can use the issue tracker as well or write a mail to slext-team@googlegroups.com. If you have a great idea for a new SLext feature, please submit them to one of the previously mentioned communication channels.

    Changelog

    version 1.0.4

    • Fixed a bug with dead tabs after lost connection to ShareLatex server or files being moved. Moved files will now be removed from the tabs when clicked, if the file was not moved, the tabs should come back alive.

    version 1.0.2 & 1.0.3

    • Reverted settings to be saved globally to match ShareLatex behaviour (tabs are still saved per project)
    • SLext will now load on projects that are accessible with shared links

    version 1.0.0

    • Rewrote SLext in TypeScript
    • Settings are now saved per-project instead of globally
    • Tabs are also saved per-project, so that tabs does not disappear when opening other project
    • Added new command for wrapping selections in LaTeX command
    • Removed setting for disabling plugins, plugins will instead offer sections in settings menu for disabling features
    • Added drag-and-drop to tabs
    • Added a changelog
